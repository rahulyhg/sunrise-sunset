== Description ==

**English**
Best Google Adsense is a free plugin and open source software that automatically insert Google Adsense ads.This plugin is compatible with the widget system.

To make money with Google Adsense ads you just have an account with Google Adsense.

You have full control over the ads. It is possible to control color, size, number and types of ads. You determine how many Adsense ads you want to show.

**[Download](http://downloads.wordpress.org/plugin/best-google-adsense.zip)**

Features:

* Automatically insert Google Adsense in your blog.
* **Earn money** with Google Adsense ads.
* This plugin will not work properly if used on site with adult/pornographic content. 
* The plugin is campatible with the widget system.
* Multi-lingual with Wordpress standards (.mo, .po).
* Friendly and intuitive interface.
* You're always on the control. Without any programming knowledge you configure the characteristics of the ads.
* With a simple click you change the type, quantity, size, placement and colors of the ads.
* Choose how many ads you want to display per page, and per post.

**Important**
As stated in Google Adsense program policies. AdSense publishers are not permitted to place Google ads on pages with pornographic content.
This includes photos or videos which contain full nudity or sexual activities. More information about this policy can be found in Adsense help center (https://www.google.com/adsense/support/as/bin/answer.py?hl=en&answer=105957).

3. Configure the plugin through the menu 'Settings'

== Frequently Asked Questions ==

= Posso usar este plugin apenas para exibir anúncios do Google Adsense? =

Claro, você pode usar este plugin para exibir anúncios do Google Andsense. Lembre-se no Best Google Adsense você está sempre no Controle

= O que é o Google Adsense? =

O Google AdSense é um programa gratuito que permite que editores on-line gerem receita exibindo anúncios relevantes em uma ampla variedade de conteúdo on-line.

= Como eu faço para exibir os anúncios do Google Adsense? =

Basta que você seja um afiliado do [Google Adsense](http://google.com.br/adsense) e instale este plugin.

= O plugin não funciona! =

O plugin **requer PHP 5** ou superior para funcionar. Se o seu blog está usando a versão 4, não tem como o plugin exibir os anúncios.

= Can I use  this plugin at adult/pornography sites? =

No, you can't use this plugin to show ads at adult/pornography sites. As stated in Google Adsense program policies. AdSense publishers are not permitted to place Google ads on pages with pornographic content.
This includes photos or videos which contain full nudity or sexual activities. More information about this policy can be found in Adsense help center (https://www.google.com/adsense/support/as/bin/answer.py?hl=en&answer=105957).
